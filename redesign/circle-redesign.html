Circle Page ‚Äî Design Notes
Key Changes from Current Implementation
Before
Challenge cards use cyan gradient ['#4facfe', '#00f2fe'] ‚Äî inconsistent with black/gold theme. No progress bars on challenge cards. No day indicator badges.
After
All challenge cards use linear-gradient(135deg, #D4AF37, #FFD700) gold gradient icons. Added thin progress bar showing challenge completion. Day badge pill on right side.
Before
No momentum/check-in visibility. Users can't see who checked in today without going to Social feed. No FOMO driver.
After
Added Momentum Banner (gold tint, üî• when ‚â•50% checked in) + Today's Check-ins horizontal scroll strip with gold progress rings showing each member's daily completion. Fully monochrome ‚Äî gold/black/white only.
New Features
‚Üí
Momentum Banner ‚Äî Only appears when ‚â•50% of circle members have checked in today. Shows fire emoji, count ("8 of 12"), and overlapping avatar faces of checked-in members. Drives FOMO for members who haven't checked in yet. Ties into the MOMENTUM_SYSTEM_SPEC.md logic.
‚Üí
Today's Check-ins Strip ‚Äî Horizontal scrollable row of member avatars with conic-gradient gold progress rings. Gold ring = completed actions (proportional fill). Empty ring = hasn't checked in. Shows completion fraction badge ("5/5") for perfect days. Uses conic-gradient(var(--gold) var(--progress), transparent var(--progress)) in React Native this would be an SVG circle with strokeDasharray.
‚Üí
Challenge Progress Bars ‚Äî Thin 3px progress bars on compact challenge cards showing how far through the challenge the circle is. Day badge pill on the right replaces having to calculate mentally.
‚Üí
Leaderboard Consistency Bars ‚Äî In the Community tab, each ranking row now has a mini progress bar + percentage showing consistency at a glance, not just raw points. Points shown to the right of the bar.
‚Üí
Invite Code in Settings ‚Äî Added a dedicated "Invite Code" row in settings with a "Copy" action, making it easier to share than the current alert-based flow.
‚Üí
Momentum Alerts toggle ‚Äî New notification setting for momentum milestones (when circle hits 50% or 100% check-in thresholds).
Podium Improvements
‚Üí
Added visual podium bars behind the top 3 contributors ‚Äî gold for 1st (tallest), subtle white for 2nd and 3rd. The bars create a visual "stage" effect. 1st place avatar is larger (64px vs 50px) with a gold glow shadow. All monochrome ‚Äî no silver/bronze colors.
‚Üí
Top 3 rank numbers in the community leaderboard use tiered opacity ‚Äî bright gold for #1, white-80 for #2, white-60 for #3 ‚Äî staying fully monochrome.
Overview Tab ‚Äî Section Order (Top to Bottom)
1.
Circle Hero ‚Äî Large emoji icon, description, 3 stat values (members, challenges, posts)
2.
Momentum Banner ‚Äî Conditional. Only shows when ‚â•50% checked in. Fire emoji + count + avatars.
3.
Today's Check-ins ‚Äî Horizontal scroll of member progress rings. At-a-glance daily status.
4.
Recent Activity ‚Äî Timeline with gold vertical line. Latest 3 posts (photo or text). Milestone dot for first.
5.
Top Contributors ‚Äî Podium with bars. "Full Leaderboard ‚Üí" link switches to Community tab.
6.
Active Challenges ‚Äî Compact cards with gold icon, progress bar, day badge. "See All ‚Üí" link.
7.
Members Grid ‚Äî 4-column grid, 8 shown. Admin crown badge. "View All (12) ‚Üí" link.
Community Tab ‚Äî Section Order
1.
Leaderboard ‚Äî Full rankings with rank number, avatar, name, consistency bar + percentage, points. Current user row highlighted gold.
2.
All Challenges ‚Äî Same cards as overview but full list.
3.
All Members ‚Äî Search bar + filter pills (All / Admins / Most Active) + list view with avatar, name, admin badge, join date, points.
Color Palette (Consistent Black/Gold Theme)
‚Üí
STRICTLY MONOCHROME ‚Äî Only three color families: gold, black, white. No green, no red, no blue, no orange.
‚Üí
Gold primary: #D4AF37, #E7B43A, #FFD700 ‚Äî stats, accents, active states, badges, progress rings, momentum banner
‚Üí
Gold dim: rgba(231,180,58,0.15) ‚Äî active tab bg, filter pill bg, admin badge bg, podium glow, check-in badges
‚Üí
Blacks: #000 (pure), #0A0A0A (soft), #111 (card), #1A1A1A (elevated/modal)
‚Üí
White layers: rgba(255,255,255, 0.04/0.06/0.08/0.10) ‚Äî card bgs and borders at different elevation levels
‚Üí
NO other colors ‚Äî No green, no cyan ['#4facfe', '#00f2fe'], no orange/fire, no red. Everything is gold, black, or white at varying opacities.
React Native Implementation Notes
‚Üí
Check-in progress rings: Use react-native-svg <Circle> with strokeDasharray and strokeDashoffset. Same approach as the daily page progress ring. Circumference = 2 * PI * radius. Offset = circumference * (1 - progress).
‚Üí
Momentum banner: Conditional render. Query: SELECT COUNT(*) FROM daily_reviews WHERE circle_id = X AND review_date = TODAY or track via action completions. Show when count / total_members ‚â• 0.5.
‚Üí
Check-in strip: ScrollView horizontal with showsHorizontalScrollIndicator={false}. Each avatar is a Pressable that navigates to profile.
‚Üí
Podium bars: Use LinearGradient with height varying per rank (64px / 48px / 36px). borderRadius: [8, 8, 0, 0] for top corners only.
‚Üí
Challenge progress bar: Simple View with width: `${percentage}%` and gold gradient bg inside a 3px-height container.
‚Üí
Current user highlight: Check member.user_id === user?.id and apply gold-tinted background + border for leaderboard row and member list item.
