Option 8A: "You" Tab with My View / Public View Toggle

Merge Progress + Profile into "You" tab. Add toggle at top to switch between "My View" (all goals) and "Public View" (what others see).

üîë Key Features

  • My View: Shows ALL your goals (public + private)
  • Public View: Preview what others see (private goals hidden)
  • Privacy badges: Goals tagged as "Public" or "Private"
  • Tabs: Goals, Posts, Stats
  • Hero section: Avatar, name, bio, quick stats at top

‚úÖ Pros

  • Keeps 5 tabs (frees up space for Challenges)
  • Clear privacy control with toggle
  • User can preview public view anytime
  • All "You" content in one place
  • Progress stats + identity unified
  • Simple implementation

‚ùå Cons

  • Toggle might be confusing at first
  • Merges introspective + social content
  • Loses dedicated Progress visualization
  • Users need to remember to set privacy per goal

Option 8B: "You" Tab with Separate Public/Private Tabs

Merge Progress + Profile into "You" tab. Instead of a toggle, use separate tabs: "Public", "Private", and "Stats".

üîë Key Features

  • Public Tab: Shows all public goals (what others see)
  • Private Tab: Shows all private goals (only you see)
  • Stats Tab: Shows consistency metrics, badges, weekly breakdown
  • Color coding: Public goals have green border, private have red border
  • Clear separation: Public vs private content in distinct tabs

‚úÖ Pros

  • Keeps 5 tabs (frees up space for Challenges)
  • Very clear mental model (Public vs Private vs Stats)
  • No confusion about what's visible to others
  • Easy to manage privacy (just change tabs)
  • Clear visual separation of content
  • No need to filter/toggle

‚ùå Cons

  • Can't see all goals at once
  • Need to switch tabs to see different goals
  • Extra navigation layer (tab within tab)
  • Might feel fragmented for users with many goals
